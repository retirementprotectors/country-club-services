# Country Club Services

Brooks Millang's neighborhood service business — window washing, lawn / yardwork, and car detailing.

## What's in this repo

| Path | Purpose |
|------|---------|
| `index.html` | Public landing page (services + booking + about) |
| `services/` | Per-service detail pages |
| `flyers/` | Print-ready door-knock flyers with QR codes |
| `crm/` | Door-knock CRM (neighborhood database + booking tracker) |
| `data/` | Neighborhood-house database (CSV) |
| `assets/` | Brand wordmark + photography placeholders |

## Quick start

This is a static site. Open `index.html` in a browser.
